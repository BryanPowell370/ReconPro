# ReconPro
It's an optimization tool for running multiple pen-testing tools at once.

## Requirements
```bash
pip install os-sys
pip install optparse-pretty
pip install numpy
pip install termcolor
```
## Installation
```bash
git clone https://github.com/LaphingMan/ReconPro.git
```
